# Polygon Calculator Frontend

This frontend is a simple HTML, CSS, and JavaScript application that provides a user interface for the Polygon Calculator.

## Features

-   A simple and clean user interface to calculate the area of a square.
-   Displays the result of the calculation.
-   Displays a list of the 5 most recent calculations.
-   Displays the total number of calculations.
-   The history is automatically refreshed every 30 seconds.

## How to Use

1.  **Backend Requirement:** Make sure the [backend server](../backend) is running.
2.  Open the `index.html` file in your web browser.

That's it! You can now enter the side length of a square and click "Calculate Area" to see the result.
